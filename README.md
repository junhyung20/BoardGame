# BoardGame Server Skeleton

This repository contains a simple Java structure for the server/domain side of a 4-player online mobile board game.

Start with these docs:

- [Code Structure](docs/CODE_STRUCTURE.md)
- [Implementation Plan](docs/IMPLEMENTATION_PLAN.md)
- [Team Assignment](docs/TEAM_ASSIGNMENT.md)

Firebase Realtime Database is wired into the storage layer. Add the real `app/google-services.json` from Firebase Console before syncing or running the app.
